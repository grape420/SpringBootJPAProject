# SpringBoot JPA Project

> SpringBoot + JPA + QueryDSL을 활용한 게시판 예제 프로젝트입니다.  
> 현재 댓글 기능은 개발 중이며, 향후 Docker 기반 구성도 예정되어 있습니다.

---

## 🛠 사용 기술 스택

- **Spring Boot**
- **Spring Data JPA**
- **QueryDSL**
- **Thymeleaf**
- **MySQL**

---

### 🔐 회원 관리
- 회원 등록
- 회원 수정
- 회원 조회
- 회원 삭제

### 📝 게시판 기능

- 게시글 수정
- 게시글 조회
